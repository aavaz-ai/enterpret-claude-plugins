# Changelog

## 5.3.0 — 2026-03-15

Initial public release.

### Commands
- `/start` — First-time setup with automatic onboarding
- `/find [topic]` — Quick feedback lookup with themes, sentiment, and quotes
- `/analyze [topic]` — Deep analysis with trends, subthemes, and patterns
- `/rootcause [issue]` — Root cause investigation with severity and evidence chains
- `/explore [category?]` — Interactive taxonomy browser
- `/brief [type?]` — Shareable summaries (weekly, account, executive, team)
- `/report [archetype?]` — Branded report generation (docx, pptx, html)

### Agents
- **wisdom-expert** — General-purpose feedback research
- **report-builder** — Long-form report generation

### Skills
- wisdom-kg, evidence-synthesis, report-engine, user-context, onboarding
